# Aimlab Vue

## Description
Simple game like Aimlab. The goal of the game is to burst as many balloons as possible as quickly as possible

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
